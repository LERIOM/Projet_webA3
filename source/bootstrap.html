Bienvenue sur le site web de notre projet outils complexes !
Ces 4 dernières semaines nous avons travaillé sur une base de données de bateaux naviguant dans le golfe du Mexique.
Nous avons nettoyé cette base de données et avons ensuite créé des scripts Python afin de prédire le type de bateau, la trajectoire des bateaux et enfin de prévoir le cluster auquel il appartient.

Ce site web sera dédié à la présentation de nos résultats.
Bonne découverte !
